Class: AbstractConstraintModel

AbstractConstraintModel

new AbstractConstraintModel()

This class encapsulates changes caused by ConstraintPanels to RestSource. Any Constraint Panel must implement function called getConstraintModel which would return this object. At any momemnt in time, RestSource could collect constraint models from Constraint Panels subscribed to it and build the final ConstraintModel via intersection calls.

MySource = new Framework.RestSource.extend({
